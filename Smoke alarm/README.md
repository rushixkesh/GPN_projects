# Smoke Detector Alarm

## INTRODUCTION

This mini project is the made to detect smoke and trigger alarm to alert human beings 

## WORKING

IF input is above a set limit then it will turn on the buzzer

## HARDWARE REQUIREMENTS

- Arduino uno
- Smoke sensor
- Buzzer
- Jumper wires
